# Architecture

CareerPilot AI follows a three-layer architecture.

## Frontend

React + TypeScript

Responsible for:

- User Interface
- Form Validation
- API Requests
- Displaying Predictions

---

## Backend

FastAPI

Responsible for:

- Request Validation
- Data Transformation
- Model Loading
- Prediction
- Response Formatting

---

## Machine Learning Layer

Random Forest Classifier

Responsible for:

- Prediction
- Confidence Score
